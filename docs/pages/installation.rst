Installation
------------
Install via ``pip``:

.. code-block:: bash

    pip3 install signatory


Alternatively install via ``git``:

.. code-block:: bash

    git clone https://github.com/patrick-kidger/signatory.git
    cd signatory
    pip install .

Requires `PyTorch <http://pytorch.org/>`__. Tested with version 1.0.1.
